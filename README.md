## Project Overview
This project presents an end-to-end Retail Sales Performance Analysis using Power BI.
It focuses on sales trends, profitability, forecasting, and actionable business insights.

## Tools & Technologies
- Power BI
- DAX
- Power Query
- GitHub

## Dataset
- SuperStore Sales Dataset

## Key Dashboards
- Executive Sales Dashboard
- Sales Forecast (15 Days)
- Product Performance & Insights
- Profit Flow Analysis
- Q&A Summary Page

## Key Metrics
- Total Sales: 1.57M
- Total Profit: 0.18M
- Total Orders: 22K
- Profit Margin: 11.19%

## Business Insights
- West region contributes the highest sales and profit
- Copiers generate the highest profit (42.77K)
- Tables and Supplies are loss-making sub-categories
- Sales show upward trend with seasonal spikes
- Profit distribution includes both gains and losses
